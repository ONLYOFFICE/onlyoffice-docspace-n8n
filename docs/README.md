# ONLYOFFICE DocSpace n8n Nodes Documentation

Documentation for ONLYOFFICE DocSpace integration with n8n workflow automation platform.

## Documentation Structure

### [Credentials Setup](./credentials/README.md)

Configure authentication for ONLYOFFICE DocSpace nodes. Supports API Key, Basic Auth, OAuth2, and Personal Access Token.

### [ONLYOFFICE DocSpace Node](./app-node/README.md)

Main node for interacting with ONLYOFFICE DocSpace API:

- [File Operations](./app-node/file-operations.md)
- [Folder Operations](./app-node/folder-operations.md)
- [Room Operations](./app-node/room-operations.md)
- [User Operations](./app-node/user-operations.md)

### [ONLYOFFICE DocSpace Trigger Node](./trigger-node/README.md)

Webhook-based trigger node for automating workflows based on DocSpace events.
